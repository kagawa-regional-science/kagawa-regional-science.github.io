# Latest Seminar

# Regional Complexity Workshop 2026

日時： 2026年2月21日（土）〜22日（日）
（1日目：午後、2日目：午前を予定）

場所： 香川大学（高松市）


本ワークショップでは、地域経済の構造と動学を理解するための多様な分析手法に関する研究発表を募集します。産業連関分析、エージェントベースモデル、ネットワーク科学、都市・交通経済学など、実証・理論・シミュレーションを問わず幅広いトピックを歓迎します。

【募集トピック例】

- 産業連関分析： 産業間取引データを活用した経済波及効果の分析、地域経済の活性化に向けた産業連携、経済政策の効果分析と予測など。
- エージェントベースモデル： 経済システムのシミュレーション、個人や企業のネットワーク行動と経済的影響など。
- 都市・交通経済学： 都市圏における経済活動の集中と拡散、交通ネットワークの効率化と経済効果、渋滞・公共交通政策の影響、都市間輸送ネットワークの最適化など。
- ネットワーク科学： 企業間取引ネットワークと経済成長、ソーシャルネットワークと意思決定、経済圏の形成とネットワーク発展、インフラネットワークの可視化と分析など。
